Подготовка установочной среды⌗

Получение свежего образа archlinux⌗

Перейдите на сайт https://archlinux.org/download, выберете ближайшее зеркало и скачайте .iso образ

Либо с помощью http клиента:

curl -O https://geo.mirror.pkgbuild.com/iso/latest/archlinux-x86_64.iso

Создание установочного носителя⌗

Далее создадим носитель с которого мы будем грузиться в среду установки. Используя команду dd, перенесите iso образ системы на вашу флэшку или портативный диск.

dd if=<iso_image_path> of=/dev/<installation_device>

Также можно воспользоваться программами Rufus, WoeUSB, прямым назначением которых является создание таких носителей.

Подготовка системы⌗

Подготовка системы может выполняться как в домашней системе, так и в самой установочной среде, в которой собраны необходимые утилиты.

Выясните программное обеспечение вашей материнской платы. От этого зависит какой тип форматирования диска поддерживает ваша система.

Форматирование жесткого диска⌗

На современных UEFI прошивках, настоятельно рекомендуется использовать GPT формат. На устаревших BIOS системах, используйте MBR разметку.

Для операций с дисками можно воспользоваться такими программами как parted (gparted для GUI), gptdisk, cfdisk, gnome-disks

Запустите программу и отредактируйте нужные свойства Файлы носителя могут быть под именем sd, hd, nvme в зависимости от типа + порядковый номер устройства

parted /dev/sdX

Желательно создать как минимум три раздела с точками монтирования:

  • EFI загрузчика
  • подкачки
  • корня системы

В случае MBR таблиц, можно обойтись двумя, соответственно без EFI раздела

После форматирования диска, нужно отформатировать сами разделы в правильные файловые системы. Делается это с помощью команд набора mkfs

Для EFI разделов предпочтительной файловой системой является FAT32: mkfs.fat -F 32 /dev/sda1

Для основной самой системы можно использовать такие фс как современную btrfs, проверенную ext4 или mkfs.ext4 /dev/sda3

И раздел подкачки

mkswap /dev/sda2
swapon /dev/sda2

Стабильное интернет соединение⌗

В связи с тем что установочные пакеты передаются по сети, последнее но не маловажное условаие обеспечить себя хорошим интернет соедеинением.

Как правиило в образе уже запущены все необходимые утилиты для быстрой настройки проводных подключений.

Установка⌗

Монтируем корневой раздел на который распакуем основные компоненты

mount /dev/sda3 /mnt

Синхронизируем пакетный менеджер для получения свежих версией пакетов и базы данных

pacman -Syu

Устанавливаем базовые пакеты, ядро, загрузчик и необходимые драйверы

pacstrap -i /mnt base linux grub linux-firmware

Настраиваем нужный образ загрузчика grub При UEFI:

pacstrap -i /mnt efibootmgr
mount --mkdir /dev/sdXN /mnt/boot/esp
grub-install --target=x86_64-efi --efi-directory=/mnt/boot/esp --boot-direcory=/mnt/boot --bootloader-id=ArchLinux

При BIOS: grub-install --target=i386-pc /dev/sdX

Когда все базовые пакеты и загрузчик распакуются, переходим в установленную систему для последующей конфигурации

arch-chroot /mnt

Конфигурация⌗

Пользователи⌗

Задаем пароль для суперпользователя

passwd

Добавляем нового пользователя и задаем ему пароль

useradd -m <username> -p <password>

Локализация⌗

Используя всю мощь systemd утилит , меняем язык и время

localectl set-locale LANG=ru_RU.UTF-8
nano /etc/locale.genlocale-gen
timedatectl set-timezone Europe/Moscow

Также можно включить синхронизацию времени по сети

timedatectl set-ntp 1

Сеть⌗

Задаем параметры домена

hostnamectl set-hostname <hostname>

И настраиваем DNS записи… (Cloudflare и Google DNS)

echo “1.1.1.1\n8.8.8.8” >> /etc/resolv.conf

Загрузчик⌗

Создаем файл конфигурации загрузчика GRUB2 с помощью вспомогательной утилиты: grub-mkconfig -o /boot/grub/grub.cfg

Завершение⌗

Отмонтируем все точки umount /mnt*

Для проверки UEFI загрузки используйте efibootmgr
